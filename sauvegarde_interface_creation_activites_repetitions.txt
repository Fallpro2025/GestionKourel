SAUVEGARDE - INTERFACE DE CRÉATION D'ACTIVITÉS AVEC RÉPÉTITIONS
===============================================================

Date: 30/09/2025
Projet: Gestion Kourel

DESCRIPTION
===========
Amélioration de l'interface de création d'activités pour permettre la création 
d'activités avec répétitions automatiques en utilisant des horaires prédéfinis.

FONCTIONNALITÉS AJOUTÉES
========================

1. INTERFACE DE CRÉATION AMÉLIORÉE
----------------------------------
- Choix entre "Activité simple" et "Avec répétitions"
- Section répétitions qui s'affiche/masque selon le choix
- Design cohérent avec le système existant

2. HORAIRES PRÉDÉFINIS
----------------------
- Mardi Soir: 20h30 - 22h30
- Dimanche Midi: 12h00 - 15h00  
- Jeudi Goudi: 20h45 - 21h15
- Horaires personnalisés: possibilité de définir manuellement

3. AUTO-SÉLECTION INTELLIGENTE
------------------------------
- Sélection automatique des jours selon le type d'activité:
  * Répétition: Mardi + Dimanche
  * Goudi Aldiouma: Jeudi
- Remplissage automatique des horaires selon la sélection

4. GÉNÉRATION AUTOMATIQUE
-------------------------
- Création de l'activité de base
- Génération automatique des répétitions selon:
  * Jours sélectionnés
  * Période définie (date début/fin)
  * Horaires prédéfinis ou personnalisés
- Redirection vers la page des répétitions après création

FICHIERS MODIFIÉS
=================
- resources/views/activites/create.blade.php

FONCTIONNALITÉS JAVASCRIPT
==========================

1. Gestion de l'affichage des sections
- Affichage/masquage de la section répétitions
- Affichage/masquage des horaires personnalisés

2. Auto-sélection intelligente
- Sélection automatique des jours selon le type d'activité
- Remplissage automatique des horaires prédéfinis

3. Gestion des formulaires
- Création d'activité simple (flux existant)
- Création d'activité avec répétitions (nouveau flux)

4. Génération des répétitions
- Appel à l'API de génération automatique

UTILISATION
===========

1. Créer une activité simple
- Sélectionner "Activité simple"
- Remplir les champs habituels
- Soumettre le formulaire

2. Créer une activité avec répétitions
- Sélectionner "Avec répétitions"
- Choisir un horaire prédéfini ou personnalisé
- Sélectionner les jours de répétition
- Définir la période de génération
- Soumettre le formulaire

HORAIRES PRÉDÉFINIS DISPONIBLES
===============================

1. Mardi Soir (20h30 - 22h30)
- Pour les répétitions régulières du mardi
- Auto-sélection du mardi

2. Dimanche Midi (12h00 - 15h00)
- Pour les répétitions du dimanche
- Auto-sélection du dimanche

3. Jeudi Goudi (20h45 - 21h15)
- Pour les prestations Goudi Aldiouma
- Auto-sélection du jeudi
- Durée plus courte (30 minutes)

4. Horaires personnalisés
- Possibilité de définir manuellement
- Champs heure début/fin
- Flexibilité totale
