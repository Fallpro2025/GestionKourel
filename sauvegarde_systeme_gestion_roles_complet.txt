UTILISATION
===========

1. ACCÈS À L'INTERFACE
   - URL: http://127.0.0.1:8000/roles
   - Interface moderne avec statistiques

2. CRÉER UN RÔLE
   - Cliquer sur "Nouveau Rôle"
   - Remplir le formulaire avec nom, description, niveau
   - Sélectionner les permissions
   - Valider

3. MODIFIER UN RÔLE
   - Cliquer sur l'icône "Modifier" dans la liste
   - Modifier les informations
   - Sauvegarder

4. VOIR LES DÉTAILS
   - Cliquer sur l'icône "Voir" dans la liste
   - Affichage complet avec membres associés

5. SUPPRIMER UN RÔLE
   - Cliquer sur l'icône "Supprimer"
   - Confirmation requise
   - Impossible si des membres utilisent le rôle

INTÉGRATION AVEC LE SYSTÈME
==========================

- Les rôles sont automatiquement disponibles dans les formulaires de membres
- Relation bidirectionnelle Role ↔ Membre
- Comptage automatique des membres par rôle
- Système de permissions extensible
- Compatible avec le système de gestion des membres existant
